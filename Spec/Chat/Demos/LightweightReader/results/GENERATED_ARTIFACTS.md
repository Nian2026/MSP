# Generated Artifacts

The PNG files in this directory are generated UI conformance evidence for the
LightweightReader `.chat` demo.

- `desktop-ui-conformance.png`
- `mobile-ui-conformance.png`

They are produced by the Playwright-backed `npm run test:ui` workflow described
in `Spec/Chat/Demos/LightweightReader/README.md`. They are not third-party
artwork and should be regenerated when the demo UI fixture or renderer changes.
