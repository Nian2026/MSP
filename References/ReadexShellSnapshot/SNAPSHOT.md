# Readex Shell Snapshot

This directory is a public index for a local-only reference snapshot used as a
shell behavior reference.

Snapshot date: 2026-06-27

Purpose:

- Keep the mature Readex shell implementation available for MSP extraction.
- Preserve original project-relative paths for easier comparison.
- Avoid modifying the original Readex project while designing Model Shell Proxy.
- Keep the extracted source subtree ignored by Git and out of the publishable
  repository surface.

Included scope:

- shell service implementation files
- shell runtime adapter files
- shell bridge, harness, panel, and transcript support files
- shell tests, parity scripts, and brokered shell patches

This snapshot is reference material, not the first MSP source layout.
