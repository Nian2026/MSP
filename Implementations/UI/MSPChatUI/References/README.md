# Local References

This folder is for local-only source snapshots used while extracting renderer
behavior. Snapshot contents are ignored by Git so the MSP SDK stays clean for
public release.

Before any copied file becomes part of the SDK, move the curated version into
`Renderers/Default/` and add the required provenance, license, and third-party
dependency notes next to the runtime asset.

Suggested local snapshot name:

```text
AIReadingReadexModeSnapshot/
```
