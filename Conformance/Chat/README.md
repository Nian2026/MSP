# `.chat` Conformance

This directory contains executable or retained evidence for the `.chat` spec.

`CodexCliValidation/` is the source-backed validation package for comparing
MSP `.chat` and agent-runtime behavior against Codex CLI/backend behavior. It
is evidence for conformance work, not public spec prose.

Public `.chat` contract docs remain under `Spec/Chat/`. Swift validator code
remains under `Implementations/Swift/Sources/MSPChat/` and
`Implementations/Swift/Sources/MSPChatValidatorCLI/`.
