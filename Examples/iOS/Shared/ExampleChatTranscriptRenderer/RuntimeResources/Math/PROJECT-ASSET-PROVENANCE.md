# Project Asset Provenance

`legacy-spinner.apng` is a project-local UI asset for the shared ExampleChatTranscriptRenderer
loading indicator. It is shipped with the shared
ExampleChatTranscriptRenderer runtime resources and exposed to the public iOS
examples through their vendor symlinks.

If this file is replaced with third-party artwork or generated material, update
this provenance note and the root `NOTICE` before publishing the source tree.
